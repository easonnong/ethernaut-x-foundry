# ethernaut-x-foundry

## Ethernaut puzzles solved & tested with foundry.

**Ethernaut**

https://ethernaut.openzeppelin.com/

**Foundry**

https://github.com/foundry-rs/foundry

## Setup

```
foundryup
forge install
forge test
```
